Settings

Sort by Keys
Alphabetically sort all object keys when saving or when opening the Raw view.
Hide Action Bar
Hides the toolbar at the top of the editor. All actions remain available via keyboard shortcuts and the Command Palette (Ctrl+K).
Type-Safe (Strict) Mode
Enforces explicit JSON types. Each value is locked to its type (string, number, boolean, null). In Free mode, values are parsed as raw JSON and can be any valid expression.
Save in Browser (Local Storage)
Automatically persists your data to the browser's local storage. Your work is restored when you reopen the page. Disable to avoid overwriting stored data.
