Paste to Markdown

Convert rich text from any source into clean Markdown.

How it works

  1. Find the text to convert (e.g., in another browser tab)
  2. Copy it to the clipboard — Ctrl+C or ⌘C
  3. Paste it into this window — Ctrl+V or ⌘V
  4. The converted Markdown will appear below!
